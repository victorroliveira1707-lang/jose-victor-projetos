# Checklist de Bolso — Parkinson Internado

> Uso rápido à beira do leito. Para detalhes, consultar o protocolo completo.

---

## ADMISSÃO — fazer nas primeiras 2 horas

- [ ] Marcar prontuário e pulseira: **PARKINSON — NÃO SUSPENDER MEDICAÇÃO**
- [ ] Levantar lista completa de medicações com **horários exatos**
- [ ] Conferir se há fármacos de risco na prescrição (ver lista abaixo)
- [ ] Acionar fonoaudiologia (avaliar disfagia)
- [ ] Acionar fisioterapia
- [ ] Aplicar Escala de Morse (queda) e Braden (úlcera)
- [ ] Se H&Y ≥ 3 ou DP avançada → acionar neurologista

---

## MEDICAÇÃO — regras de ouro

- [ ] **Não suspender** levodopa, agonistas, rasagilina, entacapona
- [ ] Manter os **mesmos horários** da rotina domiciliar (±30 min)
- [ ] Se NPO ou disfagia grave → instalar SNG para medicar
- [ ] Se SNG impossível → rotigotina patch transdérmico
- [ ] Levodopa: administrar 30–60 min **antes** das refeições

---

## FÁRMACOS PROIBIDOS

| Nunca usar | Por quê |
|-----------|---------|
| Metoclopramida | Bloqueia D2 → piora motora |
| Haloperidol | S. neuroléptica maligna |
| Risperidona / Olanzapina | Bloqueio D2 |
| Proclorperazina | Bloqueio D2 |
| Levosulpiride | Bloqueio D2 |

**Antiemético seguro:** Ondansetrona  
**Antipsicótico seguro (se inevitável):** Quetiapina 12,5–25 mg/noite

---

## ALERTA: SÍNDROME DE RETIRADA DOPAMINÉRGICA (DRS)

**Suspeitar se:**
- Rigidez muscular acentuada de forma aguda
- Febre sem foco infeccioso claro
- Agitação, delirium, confusão
- Instabilidade autonômica (taquicardia, hipertensão, diaforese)

**Conduta imediata:**
1. Restabelecer dopaminérgico imediatamente (oral ou SNG)
2. Hidratação IV + monitoração cardíaca
3. Acionar neurologista URGENTE
4. UTI se instabilidade hemodinâmica

---

## COMPLICAÇÕES — vigiar todo dia

- [ ] Constipação → polietilenoglicol se >2 dias sem evacuar
- [ ] Hipotensão ortoestática → medir PA deitado e em pé
- [ ] Delirium hipoativo → investigar ITU, constipação, hipóxia, DRS
- [ ] Risco de aspiração → cabeceira 30–45°, fisioterapia respiratória
- [ ] TVP → mobilização precoce, HBPM se risco alto
- [ ] Quedas → cama baixa, grades, supervisão em deambulação

---

## DELIRIUM — primeira linha não-farmacológica

- Reorientar frequentemente (data, local, horário)
- Família à beira do leito
- Evitar procedimentos noturnos desnecessários
- Óculos e aparelho auditivo disponíveis
- **Se agitação grave:** Quetiapina 12,5 mg (nunca haloperidol)

---

## DISFAGIA

| Grau | Dieta |
|------|-------|
| Grave | NPO → nutrição enteral (SNG/PEG) |
| Moderada | Pastosa homogênea + líquidos espessados |
| Leve | Alimentos macios, postura: queixo baixo |

---

## FISIOTERAPIA

- Iniciar no **1º dia** de internação
- Sessões idealmente **30–60 min após levodopa** (fase ON)
- Mínimo 2 sessões/dia
- Monitorar PA antes e após mobilização

---

## CRITÉRIOS PARA ALTA

- [ ] Causa da internação resolvida
- [ ] Medicação dopaminérgica tolerada via oral
- [ ] Sem sinais de DRS ativa
- [ ] Plano alimentar estabelecido
- [ ] Cuidador/família orientados
- [ ] Encaminhamento para neurologia agendado
- [ ] Documento de alta com horários exatos das medicações
